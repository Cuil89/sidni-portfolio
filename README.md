# Sidni Ilma Neo Cyber Portfolio

Personal profile/portfolio interaktif untuk Sidni Ilma dengan tema neo-brutalism, cyber event, dan animasi lanyard ID card.

## Cara menjalankan

```bash
npm install
npm run dev
```

## Struktur aset

Simpan aset di folder `public/assets` supaya bisa dipanggil langsung dari React:

- `public/assets/sidni-profile.jpeg` untuk foto hero dan ID card.
- `public/assets/sidni-cv.pdf` untuk tombol Download CV.
- `public/assets/lanyard-invofest-2025.pdf` untuk referensi lanyard.
- `public/assets/template-web-prof.mp4` untuk background motion di hero.

Kalau nama file diganti, update juga path di `src/data/profile.js`.
